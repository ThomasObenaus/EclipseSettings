List of useful eclipse plugins:

1. EclEmma:
- tool to visualize code-coverage of unit-tests
- eclipse-updatesite: http://update.eclemma.org/

2. Findbugs
- static code analysis to look for bugs in Java code
- eclipse-updatesite: http://findbugs.cs.umd.edu/eclipse

3. Fat-Jar
- The Fat Jar Eclipse Plug-In is a Deployment-Tool which deploys an Eclipse java-project into one executable jar.
- eclipse-updatesite: http://kurucz-grafika.de/fatjar

4. ResourceBundleEditor:
- Eclipse plugin for editing Java resource bundles.
- hosted on: http://sourceforge.net/projects/eclipse-rbe/

5. EPIC
- Eclipse Perl integration
- eclipse-updatesite: http://e-p-i-c.sf.net/updates

6. PyDev
- Eclipse Python integration
- eclipse-updatesite: http://pydev.org/updates

7. EGit
- Eclipse Git integration
- eclipse-updatesite: http://download.eclipse.org/egit/updates

8. Subclipse
- Eclipse SVN integration
- eclipse-updatesite: http://subclipse.tigris.org/update_1.8.x

9. CDT
- Eclipse C/C++ integration
- eclipse-updatesite (Indigo .. eclipse-3.7): http://download.eclipse.org/tools/cdt/releases/indigo

10. Mylyn
- Task-Repository/ Management tool
- eclipse-updatesite: http://download.eclipse.org/mylyn/releases/latest

11. Eclipse tooling
- useful tools for the development of eclipse 4 plugins
- eclipse-updatesite: http://download.eclipse.org/e4/downloads/drops/S-0.12M6-201203151300/repository/
- install "E4 CSS spy" and "Eclipse E4 tools"

12. EMF
- Eclipse modelling framework
- eclipse-updatesite:  http://download.eclipse.org/releases/juno/
- install "EMF Eclipse Modelling Framework SDK" and "Ecore Tooling"
 
13. Workspace Mechanic (google)
- allows you to set preferences globally for all workspaces 
- eclipse-updatesite: http://workspacemechanic.eclipselabs.org.codespot.com/git.update/mechanic/

14. CodeRecommenders
- advanced code-completion suggestions based on the current context the cursor is located 
- eclipse-updatesite: http://download.eclipse.org/recommenders/updates/stable/e42/
